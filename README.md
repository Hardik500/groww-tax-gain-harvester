# Tax Gain Harvester

A web application to calculate optimal tax gain harvesting for India's ₹1.25L LTCG exemption.

## Features

- Upload Groww spreadsheets (MF Holdings, MF Capital Gains, Stock Holdings, Stock Capital Gains)
- View already redeemed investments for the current FY
- Calculate remaining LTCG exemption to harvest
- Get optimal recommendations for which MFs/stocks to sell and rebuy

## Usage

1. Download your spreadsheets from Groww and place them in the `data/` folder
2. Open `index.html` in a browser (or run a local server)
3. Upload your files and view recommendations

## Tax Rules (India FY 2024-25)

- LTCG Exemption: ₹1,25,000 per financial year
- Tax Rate: 12.5% on gains above exemption
- Holding Period: >12 months for LTCG
- No wash sale rules in India (can rebuy same day)
